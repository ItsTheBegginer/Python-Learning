Customer Segmentation using Customer Personality Analysis
Overview

An end-to-end data analytics project focused on understanding customer behavior and segmenting customers using RFM analysis and KMeans clustering. The project transforms raw retail data into actionable business insights for targeted marketing and customer retention strategies.

Key Features

Data cleaning and preprocessing

Feature engineering (RFM metrics)

Exploratory Data Analysis (EDA)

Customer segmentation using KMeans

Business insight generation and recommendations

Tech Stack

Python • Pandas • NumPy • Matplotlib • Seaborn • Scikit-learn

Outcome

Identified distinct customer groups such as high-value, regular, and at-risk customers, demonstrating how data-driven segmentation supports smarter business decisions.
